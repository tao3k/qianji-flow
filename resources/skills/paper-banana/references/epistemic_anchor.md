# 🏛️ Epistemic Anchor Protocol

## Mandate

Serve as the immovable foundation for all subsequent reasoning logic. Ensure absolute integrity and traceability of input facts.

## Strategic Rules

1. All input data must be sanitized and verified against the Wendao Root VFS.
2. Any state transition must be logged at the atomic level in [[Epistemic_Foundation]].
3. Sovereignty of data source is non-negotiable.
